Get rid of auto generated HTML id's for your widgets and take control!

== Description ==

It's all about semantics. If you don't like <code><div id="text-3"></code> and alikes in your front-end, this plugin enables you to change it. It works with any theme.

== Installation ==

1. Upload the whole plugin directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. That's it. All the configuring is in the usual widget admin interface.
4. Enjoy!
